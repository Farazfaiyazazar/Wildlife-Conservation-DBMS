# Wildlife Conservation DBMS
> A professional, cloud-based database management system designed to monitor wildlife species, habitats, and conservation efforts. Built with **Python (Streamlit)** and **PostgreSQL (Supabase)**.

---

## Project Overview
This system provides a centralized platform for researchers and conservationists to:
* **Track Observations:** Record wildlife sightings with population counts and dates.
* **Manage Habitats:** Monitor specific areas, their climate types, and coordinates.
* **Inventory Equipment:** Track the status of gear used in various habitats.
* **Role-Based Access:** Managed via a secure user/roles system.
